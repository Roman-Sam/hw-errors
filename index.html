Шпаргалка

Перелік основних команд:

  • git config --global user.name - встановлює глобально ім'я
  • git config --global user.email - встановлює глобально ім'я
  • git init - створює базу даних і назву по дефолту master
  • git status - сама найбільше застосовувана команда в GIT, показує дані і статус
  • git add - додає файл, можна добавити 1 або декілька файлів
  • git commit - зберігає дані, що були добавлені в індекс
  • git log - показує всю історію коммітів які є в базі даних
  • git checkout - копіює файли з історії і можна повернути файл в самий початок
  • git branch -створює,перейменовує,видаляє, гілки а також перераховує їх
  • git push - відправляє всі локальні файли в інтернет(GitHub) може питати пароль
  • git pull - викачує всі файли з віддаленного сервера
  • git merge - зєднює 1 або більше гілок в 1
  • git clone - клонує віддалений репозиторій

.gitignore - це ігнорування файлів

Команди:

  • git revert - використовується для деяких нових коммітів, щоб змінити деякі попередніх комміти
  • git reset - cкасовує зміни. А за допомогою команди git reset --hard HEAD~2 можем повернутись на 2 версії файлу пізніше
  • git rebase - Переносить зміни з однієї гілки в іншу і зєднює все в єдину гілку
  • git cherry-pick - використовується для того щоб взяти зміни, внести будь-яким коммітом,
    і спробувати застосувати їх заново у вигляді нового комміта нагорі поточної гілки
  • git stash - бере змінений стан вашої робочої директорії, і їх зберігає в сховище незавершених змін,
    які ви можете в будь-який час застосувати назад.
